# Project-C++ Practice

# Overview

I developed a console-based inventory management system in C++ to deepen my understanding of object-oriented programming, file handling, and basic data structures. The project allowed me to practice using classes, vectors, loops, functions, conditionals, and file I/O in a real-world context. By building this software, I aimed to improve my problem-solving skills as a software engineer and learn how to structure a small application for maintainability and clarity.

The software I wrote manages a list of items in a store or warehouse, keeping track of their names, quantities, and prices. Users can add new items, update quantities, delete items, search for specific items, display the full inventory, and save/load data from a file.

The purpose of this project was to explore how to implement a practical system using fundamental C++ concepts while learning to handle user input validation, error checking, and persistent storage.

Software Demo video: https://youtu.be/iCm_BqEIs5U 


# Development Environment

I developed this software using Visual Studio Code as the code editor and MinGW-w64 as the C++ compiler on Windows.

The project is written in C++ and uses the standard library, including:

<iostream> for input/output operations

<vector> for managing a dynamic list of items

<fstream> for file input and output

No external libraries were used; the project focuses on fundamental C++ features for learning purposes.
